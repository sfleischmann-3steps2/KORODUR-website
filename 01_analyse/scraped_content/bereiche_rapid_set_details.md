Rapid Set Technologie
Die Rapid Set ® Technologie basiert auf dem leistungsstarken, innovativen, schnellen und schwundneutralen hydraulischen B-CSA Zement. Seinen einzigartigen Charakter erhält der Spezialzement durch den fortschrittlichen und präzisen Herstellungsprozess. Rapid Set ® Zement ist es kein Portlandzement und es werden keine Chloride bzw. Beschleuniger hinzugegeben.
Reaktionsprozess: Vergleich Portlandzement und Rapid Set ® Zement
Der chemische Reaktionsprozess von herkömmlichem Portlandzement im Vergleich zu B-CSA Zement bindet bei einem W/Z-Wert=0,45 das zugegebene Wasser bis zu 100% chemisch.
Einzigartige Performance von Rapid Set ®
Die Grafik zeigt die deutlich verbesserte Frühfestigkeitsentwicklung eines Rapid Set ® basierten Systems, dieses speziell im Vergleich zu herkömmlichen Portlandzementen (beschleunigt). Bereits eine Stunde nach Erstarrungsende sind Druckfestigkeiten von > 20,0 N/mm² zielsicher erreichbar.
In der Grafik ist der typische Schwundverlauf der klassischen Bindemittel (CEM I und mod. CEM I) im Vergleich zur Rapid Set ® Technologie aufgeführt. Als Grenzwert für die Schwundfreiheit gelten in den USA Werte < 0,05% (US Army Corps of Engineers Shrinkage Limit). Dieses wird mit der Rapid Set ® Technologie zielsicher erreicht.

CEMENT ALL
Rapid Set CEMENT ALL ist eine Mischung aus Hochleistungs- Rapid Set Zement mit feinkörnigen Sand. Es ist schnell härtend, dauerhaft und schwundneutral. CEMENT ALL ist einsetzbar für Vergussarbeiten, Verankerungen, Ausgleichsschichten und allgemeine Betoninstandsetzungen. Nur mit Wasser anmischen und das Produkt erreicht Konstruktionsfestigkeit innerhalb einer Stunde.

Häufige Fragen
Wie wird Beton mit CEMENT ALL repariert?
Zunächst wird die Oberfläche des Betons vorbereitet, durch aufrauen und entfernen von beschädigtem oder verunreinigtem Beton. Stellen Sie sicher, dass die Betonoberfläche nicht mit Öl, Farbe, Säure, Schmutz oder anderen Materialien verunreinigt ist, welche eine gute Haftung verhindern würden. Vor Applikation vornässen der zu reparierenden Oberfläche mit Wasser, dabei Pfützenbildung vermeiden. Nach der Vorbereitung der Oberfläche, anmischen des CEMENT ALL Mörtels. Sofort nach dem Mischen CEMENT ALL auftragen und abziehen auf die gewünschte Höhe. Rapid Set bindet schnell und sondert kein Wasser ab, somit kann die Oberflächenbearbeitung sofort nach dem Verlegen erfolgen.
Dann wird die mit Rapid Set CEMENT ALL reparierte Fläche mit Wasser nachbehandelt. Ziel der Nachbehandlung mit Wasser ist, die Oberfläche von CEMENT ALL solange nass zu halten, bis diese ausreichend Festigkeit erreicht hat. Die Nachbehandlung mit Wasser sollte unmittelbar erfolgen, sobald die Oberfläche ihren feuchten Glanz verloren hat und danach noch mind. 1 Stunde andauern.
Was ist Rapid Set CEMENT ALL?
Rapid Set CEMENT ALL ist eine Mischung aus Hochleistungs- Rapid Set Zement mit feinkörnigen Sand. Es ist schnell härtend, dauerhaft und schwundneutral. CEMENT ALL ist einsetzbar für Vergussarbeiten, Verankerungen, Ausgleichsschichten und allgemeine Betoninstandsetzungen. Nur mit Wasser anmischen und das Produkt erreicht Konstruktionsfestigkeit innerhalb einer Stunde.
Handelt es sich um ein modifiziertes Portland Zement Material?
Nein. Die Hochleistungseigenschaften beruhen auf dem Rapid Set Cement.
Wie schnell kann CEMENT ALL überstrichen / beschichtet / versiegelt werden?
Rapid Set verkürzt die Wartezeit bis zur Beschichtung. Unter trockenen Bedingungen können Beschichtungen auf Wasserbasis, wie z. B. Latex-Lack aufgetragen werden, sobald das Produkt fest und trocken ist, normalerweise nach 1- 4 Stunden. Produkte auf lösemittelhaltiger Basis, sowie undurchlässige Beschichtungen, wie z. B. Ölfarbe und Epoxid können nach 16 Stunden aufgetragen werden. Die Erfahrung hat gezeigt, dass manche Beschichtungen bereits nach wesentlich kürzerer Zeit aufgebracht werden können. Beachten Sie die Empfehlungen des Beschichtungsherstellers bzgl. der Oberflächenbeschaffenheit.
Ist eine Haftbrücke notwendig?
Haftbrücken sind nicht erforderlich. Die richtige Oberflächenvorbereitung ist ausschlaggebend für einen guten Verbund. Falls eine Haftbrücke gewünscht ist, sollte sie als Ergänzung für eine gute Oberflächenvorbereitung verwendet werden, nicht als Ersatz.
Können andere Produkte mit Rapid Set Cement gemischt werden?
Das Mischen mit anderen Produkten kann zu unvorhergesehenen Ergebnissen führen und ist daher nicht empfehlenswert. Die Verwendung von für Rapid Set geeigneten Zusätzen und diversen Farben ist möglich.
Wie dick kann das Produkt auftragen werden?
0 bis 10 cm
Für welche Anwendungsgebiete kann Rapid Set CEMENT ALL verwendet werden?
Rapid Set CEMENT ALL ist multifunktional. Häufigste Anwendungsgebiete sind Betonreparaturen, Spachteln, Verankerungen, Vergießen und Ausgleichschichten.
Wie wird gemischt?
Nur Wasser hinzufügen. Empfohlen wird die Verwendung eines mechanischen Mischers z. B. ein Mörtelmischer oder ein auf eine Bohrmaschine aufgesetzter Mischer. Vor Beginn der Arbeiten sollte ausreichend Personal und geeignete Ausstattung bzw. Werkzeug vorhanden sein.
Wie wird verlegt und fertiggestellt?
CEMENT ALL kann nach herkömmlichen Methoden verlegt und auf die gewünschte Höhe abgezogen werden. Rapid Set bindet schnell und sondert keine Flüssigkeit ab, sodass das Finish unmittelbar nach der Verlegung erfolgen kann. Für schnelles Arbeiten organisieren Sie ausreichend Personal und geeignetes Werkzeug. Bei flächigen Arbeiten, verlegen der vollen Einbaustärke in einem Vorgang, horizontal vorangehen. Nicht in einzelnen Schichten verlegen.
Wie wird CEMENT ALL nachbehandelt?
Alle Rapid Set Verlegungen werden mit Wasser nachbehandelt. Ziel der Nachbehandlung mit Wasser ist, die Oberfläche von CEMENT ALL solange nass zu halten, bis diese ausreichend Festigkeit erreicht hat. Die Nachbehandlung mit Wasser sollte unmittelbar erfolgen, sobald die Oberfläche ihren feuchten Glanz verloren hat und danach noch mind. 1 Stunde andauern.
Wie kann die Verarbeitungszeit verlängert werden?
Die Verarbeitungszeit kann verlängert werden indem man die Mischtemperatur senkt oder Verzögerer zugibt. Die Temperatur kann gesenkt werden, indem man das Produkt kühl hält und gekühltes Mischwasser verwendet. Durch Zugabe von Set Control aus unserer Concrete Pharmacy wird die Verarbeitungszeit verlängert.
Kann Farbe verwendet werden?
Ja. Die Farbe von Rapid Set CEMENT ALL kann durch die Verwendung von trockenen Pulver-Pigmenten verändert werden, welche für die Verwendung in Zement Mischungen konzipiert sind.
Wie wird die Oberfläche vorbereitet?
Aufrauen der Oberfläche und entfernen von beschädigten und verschmutzen Beton. Stellen Sie sicher, dass die Betonoberfläche nicht mit Öl, Farbe, Säure, Schmutz oder anderen Materialien verunreinigt ist, welche eine gute Haftung verhindern würden. Vor Applikation vornässen der zu reparierenden Oberfläche mit Wasser, Pfützenbildung vermeiden.
Wie viel wird benötigt?
Ein Sack Rapid Set Cement All ergibt ca. 13 l fertigen Mörtel. Für spezifische Details zur Verlegung kontaktieren Sie bitte den Hersteller.
Kann es für Nassverlegungen verwendet werden?
Ja. Rapid Set CEMENT ALL ist hergestellt mit Rapid Set Cement – dies ist ein hydraulischer Zement. Hydraulische Zemente binden unter Wasser ab.
Wie lange ist CEMENT ALL haltbar?
Bei trockener Lagerung in unbeschädigter Verpackung beträgt die Lagerfähigkeit von Rapid Set CEMENT ALL 9 Monate.
Wird Rapid Set CEMENT ALL durch die Temperatur beeinträchtigt?
Kalte Temperaturen werden den Erhärtungsverlauf von Rapid Set verlangsamen, warme Temperaturen beschleunigen. Größere Temperaturschwankungen werden sich deutlicher auswirken.

DOT Europe CONCRETE MIX
DOT Europe CONCRETE MIX ist ein multifunktional einsetzbares Produkt, welches auch für die statisch relevante Betoninstandsetzung, für Reparatur und Bau von Straßen, Gehwegen, Fahrbahnen, Flug- und Landebahnen, Brücken, Parkdecks, Rampen, Treppen, Beton- und Maschinenfundamente, Schalungsarbeiten sowie Industrieböden geeignet ist. Einsetzbar auch in Nassbereichen. DOT Europe CONCRETE MIX ist von seiner Optik vergleichbar mit Portlandzement-Baustoffen und kann in ähnlicher Weise verarbeitet werden.

Reparaturbeton – hochleistungsfähig, schnellabbindend, multifunktional
DOT Europe CONCRETE MIX ist ein leistungsstarker, polymermodifizierter schnellabbindender, faserverstärkter Schnellbeton für Innen und Außen. DOT Europe CONCRETE MIX ist gem. DIN EN 1504-3 einsetzbar sowohl für statisch als auch nicht statisch relevante Instandsetzungen, ideal dort, wo schnelle Festigkeiten, Dauerhaftigkeit und schwindarmes Aushärten gefordert sind. DOT Europe CONCRETE MIX kann in Stärken von 50 – 600 mm verarbeitet werden und ist bereits nach 1 Stunde belastbar.

CONCRETE PHARMACY
SET Control verzögert die Abbindezeit des Zements und wird verwendet, um die Verarbeitungszeit von Rapid Set Produkten zu verlängern. Diese zusätzliche Zeit wird häufig bei großen, komplexen Verlegungen oder bei höheren Temperaturen benötigt, wenn der Zement naturgemäß schneller abbindet.
FLOW Control verbessert die Fließfähigkeit der Mischung erheblich. Die Konsistenz einer steifen Mischung wird flüssiger. Als Ersatz für Anmachwasser erhöht FLOW Control die Festigkeiten. Mit FLOW Control wird bei gleicher Konsistenz (Ausbreitmaß) 20 – 40 % weniger Anmachwasser benötigt.
FAST beschleunigt die Abbindezeit der Rapid Set Produkte CEMENT ALL, MORTAR MIX und CONCRETE MIX.
ASPHALT REPAIR MIX
ASPHALT REPAIR MIX ist ein hochwertiges Reparaturmaterial, das sich ideal für die schnelle, einfache und dauerhafte Beseitigung von Schlaglöchern im Asphalt eignet. Für die Reparatur von Straßenschäden, Frostaufbrüchen, Grundstücks- und Garageneinfahrten, Parkplätzen, Gehwegen, industriellen Verkehrsflächen oder auch für die Einfassung/Angleichung von Straßenkanalschächten bei kurz gehaltenen Sperrzeiten. Bereits nach ca. 30 Minuten kann der Verkehr wieder rollen. ASPHALT REPAIR MIX ist schwarz eingefärbt und hat eine hervorragende Haftung.
MORTAR MIX
MORTAR MIX ist ein multifunktionaler Schnellreparaturmörtel geeignet für Betoninstandsetzung, Reparatur von Stuck und Putz, einschichtige Außenputze, Ausgleichsschichten, Schalungsarbeiten sowie für die Reparatur von Fahrbahnen. Einsetzbar für innen und außen, auch in Nassbereichen. MORTAR MIX kann horizontal, vertikal und über Kopf verarbeitet werden.
CEMENT ALL
CEMENT ALL ist ein multifunktionaler Schnellreparaturmörtel geeignet für Betoninstandsetzung, Vergussarbeiten, Verankerung und Injektion, Spachteln, Schalungsarbeiten und Bodenbeschichtungen (ab 10 mm empfehlen wir MORTAR-MIX). Einsetzbar für innen und außen, auch in Nassbereichen.
