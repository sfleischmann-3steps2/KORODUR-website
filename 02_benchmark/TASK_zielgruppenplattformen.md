## Analyse-Framework (pro Plattform)

### 1. Kernfunktion & Nutzung
- Wofür nutzen Planer/Verarbeiter diese Plattform konkret?
- Welches Problem löst sie für die Zielgruppe?
- Wie häufig wird sie genutzt (einmalig, regelmäßig)?

### 2. Features, die unsere Zielgruppen dort schätzen
- [ ] Produktsuche / Filterung
- [ ] Technische Dokumentation (TDS, SDS, Zertifikate)
- [ ] Ausschreibungstexte / LV-Texte exportierbar
- [ ] BIM-Objekte / CAD-Downloads
- [ ] Normen- und Regelwerksverweise
- [ ] Projektgalerie / Referenzsuche
- [ ] Mengenrechner / Konfiguratoren
- [ ] Fachbeiträge / Wissen
- [ ] Community / Forum
- [ ] Direktkontakt zum Hersteller
- [ ] Personalisierung / Login-Bereich

### 3. UX-Muster, die besonders gut funktionieren
- Navigation und Findbarkeit
- Sucherfahrung
- Mobile-Nutzung
- Vertrauenssignale

### 4. Was davon könnte KORODUR auf seiner Website übernehmen?
- Direkte Übertragung möglich
- Adaptierte Version sinnvoll
- Nicht relevant für KORODUR
